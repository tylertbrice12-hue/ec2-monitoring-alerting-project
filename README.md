# EC2 Monitoring and Alerting Project

## 📌 Project Overview
Deployed an Amazon EC2 instance and implemented monitoring and alerting using AWS CloudWatch. This project demonstrates hands-on experience with cloud observability, performance monitoring, and automated alerting.

---

## ☁️ AWS Services Used
- Amazon EC2 – Provisioned and managed a virtual server  
- Amazon CloudWatch – Monitored system metrics and performance  
- Amazon SNS – Configured notifications for alerting  

---

## ⚙️ Key Implementations
- Launched and configured an EC2 instance (Amazon Linux)  
- Generated CPU load using stress testing tools  
- Monitored CPU utilization metrics in CloudWatch  
- Created CloudWatch alarms for high CPU thresholds  
- Configured SNS email notifications for alerting  
- Validated alert functionality through simulated load conditions  

---

## 🚀 Key Outcomes
- Implemented real-time monitoring of cloud infrastructure  
- Built automated alerting system for performance issues  
- Gained experience with cloud observability and troubleshooting  
- Simulated real-world cloud support scenarios  

---

## 🧠 Skills Demonstrated
- EC2 Instance Management  
- Cloud Monitoring & Observability  
- CloudWatch Metrics & Alarms  
- SNS Notification Integration  
- Troubleshooting & Performance Analysis
